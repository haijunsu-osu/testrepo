🗺️ GIS Viewer - Interactive Map Application

A comprehensive web-based GIS viewer for loading map images, calibrating coordinate systems, and adding interactive annotations.

✨ Key Features

📁 Map Loading

Upload JPEG or PNG map images with drag-and-drop interface

📍 Coordinate Calibration

Set reference points to establish geographic coordinate system

📌 Interactive Annotations

Add labeled points with descriptions and metadata

📏 Distance Measurement

Measure distances between points with geographic accuracy

🗺️ GeoJSON Support

Import and overlay GeoJSON data with automatic projection

📊 Grid Overlay

Toggle coordinate grid with zoom-dependent spacing

💾 Data Persistence

Automatic saving with export/import functionality

📱 Responsive Design

Works on desktop, tablet, and mobile devices

🎯 Quick Start Guide

  1. Load a Map: Click "Load Map Image" and select a JPEG or PNG file
  2. Calibrate: Use the Calibrate tool to set at least 2 reference points with known lat/lng coordinates
  3. Navigate: Use mouse wheel to zoom, drag to pan around the map
  4. Annotate: Switch to Annotate tool and click to add labeled points
  5. Measure: Use Measure tool to calculate distances between two points
  6. Import Data: Load GeoJSON files to overlay additional map features

🛠️ Technical Details

Built with:

  • Pure HTML5, CSS3, and JavaScript (ES6+)
  • HTML5 Canvas for high-performance rendering
  • Mathematical coordinate transformations
  • localStorage for data persistence
  • Responsive design principles

Browser Requirements:

  • Modern browser with Canvas and ES6 support
  • Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
  • JavaScript enabled
  • localStorage support
